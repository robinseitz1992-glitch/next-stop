Diese Web-App erlaubt es dir, Bars hinzuzufügen und per Knopfdruck eine zufällige Bar auszuwählen. Die ausgewählte Bar wird danach aus der Liste entfernt.

## Dateien

- `index.html` - Benutzeroberfläche
- `styles.css` - Layout und Optik
- `script.js` - Logik zum Speichern, Hinzufügen und Zufallsauswählen

## Lokaler Start

1. Öffne `index.html` im Browser.
2. Füge deine Bars ein.
3. Klicke auf "Zufällige Bar wählen", um eine Bar auszuwählen und aus der Liste zu entfernen.

## Online veröffentlichen

### Option 1: GitHub Pages

1. Erstelle ein neues Repository auf GitHub.
2. Lade die Dateien (`index.html`, `styles.css`, `script.js`, `README.md`) in das Repository hoch.
3. Aktiviere GitHub Pages in den Repository-Einstellungen.
4. Wähle den `main`-Branch und den Ordner `/` als Quelle aus.
5. Nach kurzer Zeit ist die Seite online verfügbar.

### Option 2: Einfacher Webhost

Du kannst die drei Dateien auf einen beliebigen Webhost hochladen. Jeder statische File-Host funktioniert.

## iPhone-Nutzung

### Als Web-App auf dem iPhone installieren

1. Öffne die Seite in Safari.
2. Tippe auf das Teilen-Symbol.
3. Wähle "Zum Home-Bildschirm".
4. Jetzt ist die Web-App wie eine native App verfügbar.

## Hinweis zu TestFlight

TestFlight ist für native iOS-Apps. Diese App ist eine Web-App und kann ohne TestFlight genutzt werden. Wenn du eine native iOS-App erstellen möchtest, benötigst du Xcode und ein Apple Developer-Konto.
